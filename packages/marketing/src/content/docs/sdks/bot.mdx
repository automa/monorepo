| TypeScript |   NPM    | `@automa/bot` | [GitHub](https://github.com/automa/sdk-typescript/tree/master/packages/bot) |
| Python     |   PyPI   | `automa-bot`  | [GitHub](https://github.com/automa/sdk-python/tree/master/packages/bot)     |
