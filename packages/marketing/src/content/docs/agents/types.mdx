Automa bots are grouped by how they start and how they make decisions. Use this guide to choose the right bot type for your workflow.

# Trigger Types

> How does the bot get started?

## Manual Bots

Manual bots run only when you explicitly trigger them. Use a manual bot when you want on‑demand code automation — for example, generating a pull request or running a script at the click of a button.

## Scheduled Bots

Scheduled bots run automatically on a regular cadence that you configure (daily, hourly, or custom cron schedules). Scheduled bots are ideal for routine maintenance tasks, regular report generation, or any recurring automation.

# Intelligence Modes

> How does the bot decide what to do?

## Deterministic Bots

Deterministic bots follow pre‑defined rules and logic. They produce consistent, predictable results every time, making them a great fit for repeatable tasks like code formatting, linting, or scaffolding.

## AI Bots

AI bots leverage machine learning models to generate solutions on the fly. They can write, refactor, or analyze code in more open‑ended ways, such as summarizing a codebase, generating unit tests, or drafting documentation.
